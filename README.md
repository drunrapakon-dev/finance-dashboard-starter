# Finance Dashboard Starter

Finance Dashboard Starter is a modern open-source finance and accounting dashboard template designed for developers, startups, and agencies who want to build financial tools faster.

Built with Next.js, TypeScript, Prisma, and Tailwind CSS, this starter includes a clean dashboard UI, transaction management, analytics widgets, charts, dark mode, and production-friendly architecture.

Perfect for:
- Finance dashboards
- Accounting systems
- Expense trackers
- SaaS admin panels
- Internal business tools

Features:
- Modern responsive dashboard
- Transaction management
- Income & expense overview
- Interactive charts
- Category filtering
- Search & pagination
- Prisma ORM integration
- TypeScript-first architecture
- Dark mode support
- Ready-to-customize UI

Tech Stack:
- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL / SQLite
- shadcn/ui

This free starter is designed to help developers skip repetitive setup and focus on building real products faster.

## Getting Started

Install dependencies (already done if you used `create-next-app`):

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Edit the home page at `src/app/page.tsx`. The page auto-updates as you edit the file.
